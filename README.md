# Professional CV Builder

A modern, responsive CV/Resume builder web application with real-time preview.

## Features
- Real-time preview
- PDF export
- Mobile responsive
- Customizable colors and fonts

## How to use
1. Open index.html in your browser
2. Fill in your information
3. Download as PDF
